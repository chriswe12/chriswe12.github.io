Excavator Planning Agent

VLM planning · Isaac Lab · ROS 2 action servers

This semester project built a vision-language-model agent for autonomous high-level excavator planning in Isaac Lab. The task was to clear a rock field in simulation by producing useful high-level decisions rather than directly hand-coding the full behavior.

The VLM agent interpreted the scene and selected actions, while ROS 2 action servers connected those high-level decisions to low-level control policies. This kept the language-level planning layer separate from the execution layer and made the system easier to test.

The result was a successful demonstration of a simulated excavator clearing a rock field, with the VLM agent handling task-level planning and ROS 2 providing the interface to executable robot behaviors.
